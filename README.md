# Task_manager_API

A production-ready RESTful API built with Python and Flask. This project manages user tasks with secure JWT authentication and data validation.

## Features
- **User Auth**: Secure Registration & Login using JWT (JSON Web Tokens).
- **CRUD Operations**: Full Create, Read, Update, and Delete capabilities for tasks.
- **Ownership Security**: Users can only access and modify their own tasks.
- **Validation**: Strict input validation using Marshmallow schemas.
- **Database**: Persistent storage using SQLite.

## Tech Stack
- **Backend**: Flask (Python)
- **Database**: SQLAlchemy (ORM) & SQLite
- **Security**: Flask-JWT-Extended & Werkzeug
- **Validation**: Marshmallow
